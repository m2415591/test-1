# Todo-приложение (Консольное)

Простое консольное приложение для управления списком задач (todo list). Позволяет добавлять, просматривать, отмечать как выполненные и удалять задачи. Данные сохраняются в локальный текстовый файл. Цель проекта — практика основ Python и моделирование процессов разработки.
